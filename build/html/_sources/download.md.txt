# 2. Download


You can download the latest and previous versions of GiftBTE at
 [https://bte.sjtu.edu.cn/](https://bte.sjtu.edu.cn/)
You can also download the package from the git repository as:
 [https://github.com/Gift-BTE-developer/Gift-BTE.git](https://github.com/Gift-BTE-developer/Gift-BTE.git)
If you choose the GitHub version, please use the ‘master’ branch.
